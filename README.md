## Project Overview

An interactive Hospital Patient Analytics Dashboard developed using Power BI to analyze patient demographics, hospital performance, revenue trends, treatment insights, and patient satisfaction metrics.

This dashboard helps healthcare teams monitor key performance indicators and identify trends for better decision-making.

## Key KPIs

- Total Patients
- Total Revenue
- Average Length of Stay
- Average Patient Satisfaction Score
- Readmission Rate

## Dashboard Insights

The dashboard provides analysis on:

- Patient distribution across departments
- Admission trends over time
- Revenue generated by departments
- Diagnosis-wise patient analysis
- Insurance type analysis
- Room utilization
- Patient satisfaction and readmission patterns

## Interactive Filters

Users can filter insights using:

- Gender
- Department
- Doctor
- Insurance Type
- Diagnosis
- Admission Date

## Tools & Technologies

- Power BI
- DAX
